//  G20 Development Through the ESG Lens
// Measuring Progress Beyond GDP

//  Project Overview
This project analyzes G20 countries through an Environmental, Social, and Governance (ESG)
framework to move beyond traditional development metrics like GDP.
Using World Bank data (2011–2021), it combines data analytics, visualization,
and economic interpretation to evaluate sustainable development patterns.

//  Objectives
// - Evaluate G20 development using ESG indicators
// - Identify cross-country disparities and policy gaps
// - Apply data analytics tools to real-world economic questions
// - Translate insights into policy-relevant conclusions

//  Key Indicators Analyzed
// - Access to clean cooking fuels & electricity
// - Natural resource & forest depletion
// - Agricultural land use & freshwater withdrawals
// - Income and consumption growth
// - Communicable disease burden
// - Child labor prevalence
// - CO₂ emissions & coal-based electricity
// - Coastal protection & climate resilience
// - Control of corruption
// - Economic & social rights performance

//  Methodology
// - Descriptive statistical analysis
// - Cross-sectional comparison across G20 countries
// - Panel-data based interpretation
// - Band-wise classification of indicators
// - ESG-aligned economic reasoning

//  Tools & Technologies
// - R (statistical analysis & visualization)
// - Power BI (interactive dashboards)
// - MS Excel (data cleaning & preprocessing)
// - World Bank Open Data

//  Key Insights
// - GDP growth does not guarantee ESG performance
// - Resource-rich economies face sustainability trade-offs
// - Emerging economies show progress with structural gaps
// - Governance and energy mix play a critical role in outcomes
